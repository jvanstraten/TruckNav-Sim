# Contributing to TruckNav

First off, thank you for considering contributing! It’s people like you that make the ETS2 community great.

## 🚨 Branching Strategy (Important!)
To keep our releases stable, please use the following workflow:
* **Target `develop`:** All Pull Requests must be targeted at the `develop` branch.
* **`master`:** This branch is reserved for stable production releases only. PRs sent directly to master will be closed or asked to be re-targeted.

## How Can I Contribute?

### 🌐 Languages Support
If you want to add your home language and help with the TruckNav translation,
1. Please create a **`Crowdin`** account
2. Join the project: [TruckNav Language Support](https://crowdin.com/project/trucknav)
3. Request a language or start translating!

### 🪲 Reporting Bugs
If you find a bug, please open an issue and include:
* A clear, descriptive title.
* Steps to reproduce the bug.
* Your game version and any other mods active.

### 💡 Suggesting Enhancements
We love new ideas! Please open an issue first to discuss the feature before writing code. This ensures the change fits the project goals.

## 🔀 Pull Request Process
1. **Fork** the repo and create your branch from `develop`.
2. Ensure your code follows the existing style (namings, etc.).
3. Open your PR against the **develop** branch with a clear description of what you changed.

## 🤝 Need Help?
If you have questions, feel free to join our Discord and ask there: [TruckNav Discord](https://discord.gg/C5BTXCF2jC).
